Se logro implementar una base de datos en mongo atlas, por lo tanto la base de datos esta en la nube. 
Se tiene el login en la direccion localhost:3000/ , 
se puede ver los cards de los elementos a comprar en la direccion localhost:3000/products
se tienen en esa pagina las opciones de mostrar los elementos por categoria y filtralos
se puede seleccionar una de las cards e ingresar a la pagina que genera una ruta localhost:3000/productos/:category/:id
se tiene una list para el modo administrador localhost:3000/adminproduct
en esta vista se tienen las opciones de editar o eliminar lo cual borra de la base de datos el elementos
tambien se puede agregar un nuevo producto mediantela direccion localhost:3000/adminproduct/createproduct
y crear un nuevo usuario mediante la direccion localhost:3000/admin/user
